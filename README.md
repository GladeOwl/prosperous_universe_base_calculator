# Prosperous Universe: Base Calculator

**Prosperous Universe: Base Calculator** is a simple tool for players of *Prosperous Universe* that helps calculate the **material requirements and market value** of base components.  
Given a list of buildings and their planet, the tool outputs the required materials along with estimated market prices; making it easier to plan production, trade, and logistics.

---

## What It Does

- Reads a list of base buildings and their planet assignments  
- Computes the full material list needed to build those structures  
- Fetches or uses provided market pricing data  
- Outputs a summary of required materials with estimated value  

This simplifies planning for base construction and helps optimize costs in *Prosperous Universe*.
